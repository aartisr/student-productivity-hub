# Core Workflows

This page covers practical day-to-day usage patterns.

## Workflow 1: Plan Your Day

1. Open Assignments.
2. Create tasks with title, subject, due date, and priority.
3. Open Planner and add short next actions.
4. Return to Home and confirm Workload Risk signals.

Recommended habit:

- Keep assignment items as outcomes.
- Keep planner items as executable actions.

## Workflow 2: Execute Focus Sessions

1. Open Timer.
2. Use default study/break durations or update settings in Backup panel.
3. Start study cycle.
4. Pause if interrupted, then continue.
5. Complete a cycle and let mode switch.

Result:

- Session logs become visible in Analytics.

## Workflow 3: GPA Tracking

1. Open GPA panel.
2. Enter lines as `Grade,Credits` (example: `A,3` or `B+,4`).
3. Compute GPA.
4. Repeat over time to build history.

## Workflow 4: Daily Guidance via Home

Home provides:

- Command Center (best next action)
- Workload Risk Radar
- Module Workspace personalization

Use it as your operational dashboard before jumping into deep work.

## Workflow 5: Personalize Workspace

1. Open Home.
2. In Workspace Modules:
   - enable/disable modules
   - reorder tabs
   - set default view
3. Apply preset profiles or create your custom profile.
4. Export/import custom profile packs as needed.

## Suggested Weekly Cadence

- Monday: bulk plan assignments.
- Daily: run 2 to 5 focused cycles.
- Mid-week: check analytics and adjust effort.
- Friday: export backup snapshot.
