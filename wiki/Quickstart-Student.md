# Quickstart (Student)

Use this one-page guide to get productive in under 10 minutes.

## 0 to 2 Minutes: Sign In

1. Open the app.
2. Go to Auth.
3. Click any available OAuth provider.
4. Complete sign-in and return to app.

You should now see your role as `student` in the top area.

## 2 to 4 Minutes: Capture Work

1. Open Assignments and add upcoming deliverables.
2. Open Planner and add next actions for today.
3. Keep tasks short and measurable.

## 4 to 6 Minutes: Start Focus Mode

1. Open Timer.
2. Start one study session.
3. Pause/resume if needed.
4. Complete a full study cycle.

Your session logs feed Analytics automatically.

## 6 to 8 Minutes: Check Academic Health

1. Open GPA panel.
2. Enter grade lines as `Grade,Credits`.
3. Compute and save baseline performance.

## 8 to 10 Minutes: Use Quiz Lab

1. Open Quiz Lab.
2. Create a small question bank (3 to 5 questions).
3. Run one attempt.
4. Review outcomes and upcoming review queue.

## Daily Operating Loop

- Morning: update Planner and Assignments.
- Midday: run 1 to 2 focus sessions.
- Evening: check Home recommendations and close top-priority item.

## Weekly Operating Loop

- Export backup JSON once a week.
- Review Analytics trends.
- Archive completed items and plan next week.

## Student Access Notes

- Instructor analytics and LMS connector controls are restricted.
- If you see Access Denied, your account is functioning normally for student role.

## Quick Commands

```bash
npm run dev
npm run verify
```
