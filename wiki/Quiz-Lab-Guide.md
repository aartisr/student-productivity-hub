# Quiz Lab Guide

Quiz Lab is the most advanced module in the app.

## What Quiz Lab Includes

- Question Bank Builder
- Adapter Import/Export
- Lesson Studio
- Share Exchange
- Quiz Runtime with adaptive progression
- Review scheduler
- Instructor and connector sections (role-gated)

## Build a Quiz Bank

1. Set title, subject, and difficulty.
2. Add question prompt.
3. Select question type:
   - single
   - multi
   - boolean
   - short
4. Add choices and correct answer(s).
5. Save bank.

## Import and Export Formats

Supported adapters:

- Generic JSON
- Moodle GIFT
- AIKEN
- CSV MCQ
- TSV flashcards
- QTI 2.1 XML
- QTI 2.1 package bundle

Use import dry-run feedback and warnings before publishing.

## Run an Attempt

1. Select a bank.
2. Start attempt.
3. Answer questions.
4. Submit and review score.
5. Observe review queue for upcoming spaced repetition items.

## Lessons and Sharing

Lesson Studio:

- Create content-rich lessons with tags and linked quiz IDs.

Share Exchange:

- Export lesson, quiz, or bundle payload.
- Import payload and merge into local content.

## Instructor/Admin Sections

Available only to instructor/admin roles:

- LMS connector simulation and payload diagnostics.
- Instructor analytics (blueprint, mastery bands, item analysis).

Students see explicit Access Denied panels in these areas.
