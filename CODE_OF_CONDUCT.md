# Code of Conduct

## Our Commitment

We are committed to creating a welcoming, respectful, and inclusive community for students, educators, researchers, and contributors of all backgrounds.

We value constructive collaboration, evidence-based discussion, and empathy in technical conversations.

## Expected Behavior

All participants are expected to:

- Be respectful and professional.
- Give and receive feedback constructively.
- Focus on ideas, behavior, and outcomes, not personal attacks.
- Support newcomers and foster an inclusive learning environment.
- Respect different viewpoints and experiences.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, intimidation, or discrimination.
- Insults, personal attacks, or derogatory language.
- Trolling, deliberate disruption, or bad-faith engagement.
- Publishing private information without consent.
- Any conduct that undermines safety or participation.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues, pull requests, and discussions
- Documentation and review comments
- Any community interaction representing the project

## Reporting Concerns

If you experience or observe behavior that violates this Code of Conduct, please report it by opening a private issue or contacting the repository owner:

- Owner: Aarti Sri Ravikumar
- GitHub: https://github.com/aartisr

When reporting, include as much context as possible:

- What happened
- Where and when it occurred
- Links, screenshots, or relevant evidence

## Enforcement

Project maintainers may take any action deemed appropriate, including:

- Requesting behavior changes
- Removing harmful content
- Temporarily limiting participation
- Permanently banning contributors from project spaces

Enforcement decisions prioritize community safety, fairness, and continuity.

## Good-Faith Collaboration

We recognize that disagreements happen in open-source communities. We ask all participants to assume good intent, communicate clearly, and center student benefit and scientific rigor in discussions.
